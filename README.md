# 故宫博物院数字文物模型下载器

下载`mview`文件并转成 `glb`格式，保存到系统下载目录中。

# Requirements

Python 3.8.6 or later [[download](https://www.python.org/downloads/)]

# Usage

`python .\download.py https://www.dpm.org.cn/shuziwenwu/260332.html`
